Love Matcha Stock PWA v1.1

วิธีใช้:
1) แตก ZIP
2) อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้ไปทับที่หน้าแรกของ GitHub repo Love-Matcha-Stock
3) รอ GitHub Pages deploy ประมาณ 1 นาที
4) เปิด https://prapatbase2.github.io/Love-Matcha-Stock/
5) Android: Chrome > ⋮ > ติดตั้งแอป หรือ เพิ่มไปยังหน้าจอหลัก
6) iPhone: Safari > แชร์ > เพิ่มไปยังหน้าจอโฮม

หมายเหตุ:
- เวอร์ชันนี้คือ v1.1
- เปิดหน้าเว็บได้ตอนออฟไลน์ถ้าเคยเปิดมาก่อน
- ถ้าออฟไลน์ ระบบจะปิดการบันทึก/แก้ไขข้อมูล และเตือนให้ต่ออินเทอร์เน็ตก่อน
- ข้อมูล Firebase/Firestore ไม่ถูกแคช เพื่อให้สต๊อกเป็นข้อมูลสดเสมอ


เพิ่มใน v1.1:
- Restore Backup JSON ในหน้าสำรองข้อมูล เฉพาะเจ้าของ และต้องกรอก PIN ตัวเองยืนยัน
- ระบบส่งชาให้สาขาอื่นแบบรอยืนยัน: ตัดออกจากสาขาต้นทางก่อน ปลายทางต้องกดยืนยันรับเข้า Stock จึงจะบวกเข้าสต๊อก
- หน้า Stock เรียงให้สาขาของผู้ใช้เด่นขึ้นก่อน และยังดู Stock รวมทุกสาขา/ทุกสาขาได้
- แตะตัวเลข Stock ในหน้าสาขา เพื่อไปหน้าลงรายการพร้อมเลือกสาขา/ชา/ขนาดให้อัตโนมัติ
- อัปเดตรายการชา/ขนาด/จำนวน Stock ตามไฟล์ Excel ที่แนบ รวม มัทฉะ 19 และ มัทฉะ 20
- ถ้าใช้ Firebase เดิม ให้เจ้าของเข้า “สำรองข้อมูล” แล้วกด “นำเข้า Stock จาก Excel v1.1” เพื่อเขียนข้อมูล Stock ตาม Excel ลงฐานข้อมูลจริง
